# GCD（最大公约数）

GCD 是 *Greatest Common Divisor* 的缩写，中文一般称 **最大公约数**。

## 正式定义

给定两个非负整数 \(a, b\)：

- **公约数**：能同时整除 \(a\) 和 \(b\) 的正整数
- **最大公约数**：这些公约数中最大的那一个

记作 `gcd(a, b)`。

## 简单例子

- `gcd(12, 18) = 6`
- `gcd(7, 13) = 1`
- `gcd(0, 5) = 5`

## 常用性质

- `gcd(a, a) = a`
- `gcd(a, 0) = a`
- `gcd(a, b) = gcd(b, a mod b)`（欧几里得算法核心）

## 应用：数组向右轮转的 \(O(1)\) 数学结构（环）

题目：给定整数数组 `nums`，将数组元素向右轮转 `k` 个位置（`k >= 0`）。

### 环从哪里来

设数组长度为 \(n\)。向右轮转后，原下标 `i` 会去到：

- 新下标：\((i + k) \bmod n\)

这个映射会把下标集合 \(\{0,1,\dots,n-1\}\) 分解成若干个**不相交的环**，而**环的个数**是：

- `gcd(n, k)`

### 示例

**示例 1**

- `nums = [1,2,3,4,5,6,7]`, `k = 3`
- `n = 7`，`gcd(7, 3) = 1`，所以只有 **1 个环**

下标环（从 0 出发）：

- `0 → 3 → 6 → 2 → 5 → 1 → 4 → 0`

这就是一个“大环”，从任意起点出发按这个映射做替换，最终会回到起点并覆盖所有元素。

**示例 2**

- `nums = [1,2,3,4,5,6]`, `k = 3`
- `n = 6`，`gcd(6, 3) = 3`，所以有 **3 个环**

下标环：

- `0 → 3 → 0`
- `1 → 4 → 1`
- `2 → 5 → 2`
